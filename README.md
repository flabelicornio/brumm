# Wavely — Music Player

PWA de audio para ChromeOS. Un archivo HTML, sin dependencias externas.

## Deploy en Cloudflare Pages

1. Sube esta carpeta a GitHub (o arrastra `public/` directo a CF Pages)
2. En Cloudflare Pages → Create project → Connect to Git
3. Build settings: ninguno (static site)
4. Output directory: `public`
5. Listo.

## Monetización
- Conectar Lemon Squeezy en el botón Pro del modal
- AdSense: pegar script antes de `</body>`

## Formatos soportados
MP3, FLAC, AAC, M4A, OGG, WAV, OPUS
